
Este repositorio es para material visto en la charla de Digital Transformation Saturday 2022.

"POWER BI: En busca de visualizaciones nativas más dinámicas", consiste en una charla para mostrar como sacar provecho de visualizaciones nativas de POWER BI buscando agregar dinamismo al comportamiento de las mismas.

Para poder utilizar el archivo pbix provisto, se requiere tener la base de datos AdventureWorksDW2014. Posteriormente, ejecutar el script provisto de sql "001-Add Budget To AdventureWorksDW.sql", el cuál crea una tabla para el presupuesto "Budget", y también inserta algunos datos de prueba.
